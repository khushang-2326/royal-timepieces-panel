🕰️ Royal Timepieces Panel

A modern and responsive admin panel dashboard built for managing products, users, and other backend operations for the Royal Timepieces platform. The panel is designed with clean UI, smooth interactions, and a scalable architecture suitable for e-commerce or management systems.

🚀 Features

📦 Product Management (Add, Edit, Delete products)
👥 User Management
📊 Dashboard Overview
🖼️ Image Upload & Display
🔐 Secure Authentication Flow
🎨 Modern & Responsive UI
⚡ Fast performance with optimized components

🛠️ Tech Stack

Frontend

React.js
Vite
Tailwind CSS / Custom CSS
Axios for API requests
React Router DOM

Backend

If backend is part of a separate repo, mention here accordingly.

Node.js
Express.js
MongoDB (if used)

🧩 Installation & Setup

Follow these steps to run the project locally:

1. Clone the repository
  git clone https://github.com/khushang-2326/royal-timepieces-panel
  cd royal-timepieces-panel

2. Install dependencies
  npm install

3. Start development server
  npm run dev

Your app should now be running at:

👉 http://localhost:5173
